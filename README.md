# Argalia_ears
A live whisper works on both terminal and websocket

### Description
- This project is the combination of Faster-Whisper, FastAPI, and Silero_VAD

```terminal``` version is currently not up to date, I suggest using websocket

## Websocket
- Run code: ```fastapi dev STT_api_calls.py```

## Terminal
- Run code: ```python(3) mainCall.py```
